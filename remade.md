# 熊猫秘境·生态叙事 —— MR大熊猫保护教育平台

成都数字媒体技术 毕业设计 | 挑战杯参赛项目

## 项目简介
基于成都大熊猫基地与雅安碧峰峡真实生态数据开发的 **MR混合现实沉浸式教育体验**。用户可走进虚拟熊猫栖息地，与AI熊猫互动，感受气候变化影响，并创作保护故事。

## 快速预览（推荐现在就做）

1. 打开 `WebAR_Version` 文件夹（稍后创建）
2. 用手机浏览器打开 `index.html` 即可体验网页AR版

## 技术栈
- Unreal Engine 5（MR主版本）
- WebXR / Unity（轻量演示版）
- AI大模型（角色对话 + 故事生成）

## 目录结构
- `/WebAR_Version/` → 网页快速原型（秒悟生成）
- `/UnrealProject/` → MR核心工程
- `/Docs/` → 论文、PPT、开题报告
- `/Prompts/` → AI提示词库
- `/Assets/` → 模型、贴图等
